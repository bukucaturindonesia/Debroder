# DEBRODER Soft Overlay Fix

Perbaikan ini mengubah overlay gambar agar teks tetap jelas tanpa membuat seluruh gambar menjadi gelap.

Yang diubah:
- Featured / Trending / custom card memakai soft bottom gradient saja.
- Instagram Banner memakai soft bottom gradient saja.
- Campaign Banner memakai soft bottom gradient saja.
- Page Hero memakai bottom gradient ringan, bukan overlay penuh.
- Hero Slider dibuat lebih ringan: area teks tetap terbaca, sisi gambar lain tetap lebih terang.

Prinsip desain:
- Gambar tidak diberi filter gelap penuh.
- Overlay hanya membantu keterbacaan teks.
- Teks tetap prioritas dengan drop shadow ringan dan tombol putih.

Catatan:
- File asli gambar tidak diubah.
- Pengaturan crop/focal point dari admin tetap berjalan.
