CARA MENERAPKAN

1. Extract ZIP ini.
2. Copy folder `test` ke root repository DEBRODER.
3. Izinkan replace hanya untuk:
   test/admin-order-pricing-workspace.test.ts
4. Review diff.
5. Jalankan:
   pnpm vitest run test/admin-order-pricing-workspace.test.ts
   pnpm run build

BATASAN
- Tidak ada source produksi yang berubah.
- Tidak ada database/migration yang berubah.
- Tidak ada commit, push, atau deploy yang dilakukan oleh paket ini.
