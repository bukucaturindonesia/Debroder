CARA MENERAPKAN

1. Backup repository DEBRODER terbaru.
2. Pastikan source yang akan menerima paket berasal dari Debroder(17).zip atau review setiap konflik secara manual.
3. Extract ZIP ini ke folder sementara.
4. Copy seluruh folder/file ke root repository DEBRODER dengan struktur path tetap.
5. Pilih Replace/Merge untuk file yang sama.
6. Jangan menghapus file lain.
7. Tidak ada SQL atau migration yang perlu dijalankan.
8. Review diff di GitHub Desktop.
9. Jalankan quality gates:

   pnpm vitest run test/customer-order-hub.test.ts test/order-confirmation-payment-link.test.ts test/payment-verification-workspace.test.ts
   pnpm typecheck
   pnpm lint
   pnpm test
   pnpm build

10. Owner melakukan commit dan push ke branch Preview.
11. Redeploy Preview Vercel.
